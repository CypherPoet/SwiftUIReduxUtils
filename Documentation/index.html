Installation

Xcode Projects

Using Xcode, select File -> Swift Packages -> Add Package Dependency and enter https://github.com/CypherPoet/CypherPoetReduxUtils.

Swift Package Manager

CypherPoetReduxUtils can be used through the Swift Package Manager. You can add it as a dependency in your Package.swift file:

let package = Package(
    //...
    dependencies: [
        .package(url: "https://github.com/CypherPoet/CypherPoetReduxUtils", from: "0.1.0"),
    ],
    //...
)

Then simply import CypherPoetReduxUtils wherever you’d like to use it.
